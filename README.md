# 🛒 Ecommerce Website

A web-based Ecommerce application developed using Python and Django for product management and online shopping.

---

## 🚀 Features

- Product listing
- Product images
- Category management
- Add and manage products
- User-friendly interface
- Dynamic web pages

---

## 🛠️ Tech Stack

- Python
- Django
- SQLite3
- HTML
- CSS

---

---

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/preety-m/ecommerce.git
```

### Navigate to project directory

```bash
cd ecommerce
```

### Install Django

```bash
pip install django
```

### Run the server

```bash
python manage.py runserver
```
